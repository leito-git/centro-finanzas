# Analizador de Distancia al ATH con Gráficos Interactivos

Esta aplicación de Streamlit calcula la distancia porcentual de una lista de activos respecto a su máximo de 200 ruedas e incluye visualizaciones interactivas.

### Características:
- Descarga de datos en tiempo real mediante Yahoo Finance.
- Cálculo automático de ATH de 200 ruedas.
- Gráficos interactivos con Plotly.
- Indicador RSI (14) incluido.
